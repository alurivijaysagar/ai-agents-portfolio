# AI Agents Portfolio

A collection of AI agent projects built using Python and the Google Gemini API.
Covers multi-agent systems, RAG workflows, and domain-specific AI assistants.

Built in under 2 weeks — starting from zero knowledge of LLMs and APIs.

---

## Projects

### 1. Multi-Agent Fitness Assistant
A 3-agent pipeline for personalized fitness planning.
- Agent 1 analyzes fitness goals and body metrics
- Agent 2 generates a personalized workout plan
- Agent 3 creates a diet plan based on workout output

**Concepts:** Multi-agent orchestration, context passing, prompt engineering

---

### 2. Temple Retrieval Assistant
A RAG-style assistant that answers questions about ancient temples
using structured local data injected into the model's context.

**Concepts:** Retrieval Augmented Generation (RAG), function modularization,
context injection

---

### 3. PC Recommender Multi-Agent System
A dual-agent system where one agent recommends PC builds and a second
agent critically reviews and refines the recommendation.

**Concepts:** Multi-agent review pipeline, prompt engineering, context passing

---

### 4. Car Expert AI Agent
A conversational AI assistant that helps users explore and compare
cars based on preferences and budget.

**Concepts:** Domain-specific agent design, system instruction engineering

---
