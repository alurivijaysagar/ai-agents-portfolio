# Multi-Agent Fitness Assistant

A beginner applied AI project built using Python and the Gemini API.

## Overview

This project simulates a multi-agent AI workflow for fitness planning.

The system contains 3 specialized AI agents:

* Goal Analysis Agent
* Workout Coach Agent
* Dietician Agent

Each agent has a focused responsibility and passes context to the next stage.

## Technologies Used

* Python
* Gemini API
* Prompt Engineering
* Multi-Agent Workflow Design

## Features

* Collects user fitness goals and body metrics
* Generates workout guidance
* Creates personalized diet plans
* Demonstrates context passing between agents

## Learning Outcomes

* Multi-agent orchestration
* Context injection
* Python control flow
* API integration
* Workflow-based AI design
