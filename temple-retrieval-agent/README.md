# Temple Retrieval Assistant

A beginner retrieval-based AI assistant built using Python and the Gemini API.

## Overview

This project retrieves temple information from structured dictionary data and provides curated AI-generated responses based on user queries.

## Technologies Used

* Python
* Gemini API
* Dictionaries and Nested Data Structures
* Retrieval Logic

## Features

* Searches temple data using user prompts
* Matches temple names dynamically
* Injects retrieved context into the AI model
* Modularized using Python functions

## Learning Outcomes

* Retrieval workflow design
* Function modularization
* Nested dictionary traversal
* String matching and search logic
* Prompt context injection
